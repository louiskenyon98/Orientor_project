# Orientor Fresh API

Brand new clean deployment for Orientor backend.

## Features
- ✅ FastAPI 0.104.1
- ✅ CORS configured for Vercel
- ✅ Health check endpoints
- ✅ Basic auth endpoints
- ✅ Railway deployment ready

## Deploy to Railway
```bash
railway login
railway init
railway up
```

## Local Development
```bash
pip install -r requirements.txt
python main.py
```

API will be available at http://localhost:8000
